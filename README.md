# 🌐 Antef Task 4

## 📌 Project Overview

This is a simple and responsive web project built using **HTML** and **CSS**. It displays a centered card layout with an image and a button. On clicking the button, the user is redirected to an external website.

---

## 🚀 Features

* 📷 Display of a background image inside a container
* 🎨 Stylish gradient background and button
* 🖱️ Interactive button with hover effect
* 🔗 Redirects to an external website on click
* 📱 Clean and simple UI design

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
Antef_Task4/
│── index.html

```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. The project will open in your browser

---

## 🔗 Functionality

* The button inside the container redirects users to a wallpaper website:
  👉 https://www.pexels.com/search/4k%20wallpaper/
